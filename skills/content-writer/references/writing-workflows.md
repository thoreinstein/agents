# Writing Workflows

Detailed workflow templates for different content types.

## Blog Post Workflow

1. **Outline together**
   - Discuss topic and angle
   - Structure main sections
   - Identify research needs

2. **Research key points**
   - Find supporting data
   - Gather examples
   - Collect citations

3. **Write introduction → get feedback**
   - Draft hook and context
   - Review for impact
   - Iterate until strong

4. **Write body sections → feedback each**
   - One section at a time
   - Review before moving on
   - Maintain flow between sections

5. **Write conclusion → final review**
   - Summarize key points
   - Clear call to action
   - Check overall coherence

6. **Polish and edit**
   - Tighten prose
   - Fix transitions
   - Final proofread

## Newsletter Workflow

1. **Discuss hook ideas**
   - What's the compelling angle?
   - Why now?
   - What's the payoff for readers?

2. **Quick outline (shorter format)**
   - Hook
   - 2-3 main points
   - Single call to action

3. **Draft in one session**
   - Keep momentum
   - Don't over-edit during drafting
   - Get it all down

4. **Review for clarity and links**
   - Every link works
   - Every point is clear
   - No fluff

5. **Quick polish**
   - Read aloud
   - Tighten sentences
   - Ship it

## Technical Tutorial Workflow

1. **Outline steps**
   - What's the end goal?
   - What are the prerequisites?
   - Break into logical steps

2. **Write code examples**
   - Working, tested code
   - Progressive complexity
   - Clear comments

3. **Add explanations**
   - Why, not just how
   - Anticipate questions
   - Link to deeper resources

4. **Test instructions**
   - Follow your own tutorial
   - Fresh environment if possible
   - Note any gotchas

5. **Add troubleshooting section**
   - Common errors
   - Solutions
   - Where to get help

6. **Final review for accuracy**
   - All code runs
   - All steps work
   - Version numbers current

## Thought Leadership Workflow

1. **Brainstorm unique angle**
   - What's your contrarian view?
   - What do you know that others don't?
   - What's the insight?

2. **Research existing perspectives**
   - What's the conventional wisdom?
   - Who disagrees and why?
   - Where are the gaps?

3. **Develop your thesis**
   - One clear argument
   - Defensible position
   - Memorable framing

4. **Write with strong POV**
   - Take a stance
   - Use confident language
   - Don't hedge excessively

5. **Add supporting evidence**
   - Data that supports your view
   - Examples that illustrate
   - Anticipate objections

6. **Craft compelling conclusion**
   - Restate thesis
   - Implications
   - Call to think differently
